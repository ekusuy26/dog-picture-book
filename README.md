## 制作経緯
犬を飼いたい！となってネットで検索するも、ショップのサイトでは子犬の紹介ばかり。
* 成犬をしっかりイメージできているのだろうか？
* 思ったより大きくなってこんなはずじゃなかったとならないだろか？
* 人気の犬種にだけ目が行きがちだが、こんな子もいるのに…
「子犬ベースではなく、成犬ベースで犬を探せればいい」「実際に飼っている人と繋がり、情報共有をしたい」と考え、本アプリを作成致しました。
